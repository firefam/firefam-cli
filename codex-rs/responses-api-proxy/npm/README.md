# firefam-responses-api-proxy

<p align="center"><code>npm i -g firefam-responses-api-proxy</code> to install <code>firefam-responses-api-proxy</code></p>

This package distributes the prebuilt Firefam Responses API proxy binary for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/codex-responses-api-proxy.js --help
```

Refer to [`codex-rs/responses-api-proxy/README.md`](https://github.com/firefam/firefam-cli/blob/main/codex-rs/responses-api-proxy/README.md) for detailed documentation.
